# Global Unicorn Startup Analytics (2007–2022)

## Project Overview

This project analyzes 1,037 unicorn startups across 46 countries and 33 industries to identify trends in startup growth, valuation, and geographic distribution.

## Tools Used

- Python (Pandas)
- BigQuery
- SQL
- Looker Studio
- Kaggle Dataset

## Business Questions

- Which countries have the most unicorn companies?
- Which industries generate the highest valuation?
- How has unicorn creation changed over time?
- Which companies have the highest valuations?

## Dashboard

https://datastudio.google.com/u/0/reporting/11e438ee-30a1-482b-8430-bb258a468e65/page/Vp31F

## Key Insights

- The United States accounts for over 50% of global unicorn companies.
- Fintech is the largest unicorn industry by valuation.
- Unicorn creation peaked in 2021 with 514 new companies.
- China and India are the second and third largest unicorn ecosystems globally.
